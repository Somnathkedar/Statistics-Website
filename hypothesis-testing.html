Hypothesis Testing

Hypothesis testing is used to make decisions using sample data. It helps determine whether an assumption is true or not.

Null and Alternative Hypothesis

The null hypothesis assumes no effect. The alternative hypothesis assumes there is an effect.

p-Value

If p-value is less than 0.05, the null hypothesis is rejected. Otherwise, it is not rejected.

Common Tests

t-test, z-test, chi-square test, and ANOVA are commonly used tests.
